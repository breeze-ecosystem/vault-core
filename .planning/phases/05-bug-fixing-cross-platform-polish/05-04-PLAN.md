    **Step A — Convert list screens from ScrollView to FlashList:**
    For each list screen that renders a scrollable list of items (alerts, cameras, incidents, sites, door-control), replace the `<ScrollView>` wrapper with `<FlashList>`.

    Pattern:
    ```typescript
    // BEFORE:
    import { ScrollView, RefreshControl } from "react-native";
    <ScrollView refreshControl={<RefreshControl refreshing={refreshing} onRefresh={refreshItems} />}>
      {items.map((item) => <CardComponent key={item.id} item={item} />)}
    </ScrollView>

    // AFTER:
    import FlashList from "@shopify/flash-list";
    <FlashList
      data={items}
      renderItem={({ item }) => <CardComponent item={item} />}
      estimatedItemSize={120}
      keyExtractor={(item) => item.id}
      refreshing={refreshing}
      onRefresh={refreshItems}
      ItemSeparatorComponent={() => <View style={{ height: 8 }} />}
    />
    ```

    Key settings per screen:
    - alerts.tsx: estimatedItemSize={120}, remove ScrollView
    - cameras.tsx: estimatedItemSize={140}, remove ScrollView (camera cards are taller)
    - incidents.tsx: estimatedItemSize={120}, remove ScrollView
    - sites.tsx: estimatedItemSize={120}, remove ScrollView
    - door-control.tsx: estimatedItemSize={160}, remove ScrollView (door cards include controls)
