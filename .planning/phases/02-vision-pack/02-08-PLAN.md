| Boundary | Description |
|----------|-------------|
| Mobile app → API (geofencing) | WiFi SSID and heartbeat sent over local network |
| Mobile app → API (share receive) | No-auth token access to stream URLs |
| Mobile camera/gallery → API | Face photos uploaded for whitelist registration |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-20 | Spoofing | Share link receiver | mitigate | Token verified by backend JWT validation + DB check before stream URL returned |
| T-02-21 | Tampering | WiFi SSID in geofencing | accept | SSID reported by device OS; cannot be spoofed without physical access to phone |
| T-02-22 | Info Disclosure | Camera stream on mobile | mitigate | Stream only available on local network (VISION restriction); no internet streaming |
</threat_model>
