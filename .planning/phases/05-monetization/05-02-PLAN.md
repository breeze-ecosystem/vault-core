    Step E: Create `apps/api/src/modules/license/license.service.ts`
    Follow the exact pattern from RESEARCH.md Pattern 1 (LicenseService):
    - `@Injectable()` class
    - Constructor: PrismaService, LicenseKeyManager, ConfigService
    - `generateLicense(dto: GenerateLicenseInput)`: Build LicenseClaims per D-06, call jwt.sign(claims, privateKey, { algorithm: 'RS256', issuer: 'oversight-hub' }), store License record in DB with status PENDING. Return JWT string + decoded claims (not the full token+claims as separate — return { licenseJwt: string, claims: LicenseClaims }).
    - `verifyAndActivate(jwtToken: string, orgId: string)`: jwt.verify(token, publicKey, { algorithms: ['RS256'] }), check organizationId claim matches orgId, check not expired (exp < now). Update matching PENDING license records to ACTIVE, set activatedAt. Return { status: 'active', claims }.
    - `getLicenseStatus(orgId: string)` — THE critical state machine method. Priority order per RESEARCH.md Pitfall 5:
      1. Check org exists, if not throw NotFoundException
      2. **Active license check first**: Find most recent active license (status ACTIVE, ordered by createdAt desc). If found and not expired, return active state with limits.
      3. **Grace check**: If active license found but expired, calculate grace end = expiresAt + gracePeriodDays days. If within grace, return grace state. Auto-update DB status to GRACE if it's currently ACTIVE.
      4. **Trial check**: If org.trialStartDate and org.trialEndDate exist and trialEndDate > now, return trial state with isUnlimited=true.
      5. **Auto-initialize trial**: If no trial dates set AND no active license was found, auto-set trial dates on org (trialStartDate = now, trialEndDate = now + 7 days) and return trial state. This handles orgs created before the licensing system.
      6. **Expired after grace**: If past grace, update DB status to EXPIRED, return expired.
      7. **No license at all**: return no_license state.
    - `getUsage(orgId: string)`: Count cameras and doors for the org, get license limits from getLicenseStatus. Return { cameras: { current: number, max: number (or null if unlimited) }, doors: { current: number, max: number (or null if unlimited) } }.
    - `listLicenses(orgId?: string)`: If orgId given, list for org. If not, list all (admin). Order by createdAt desc.
    - `listAllApiKeys()`: Find all LicenseApiKey records, order by createdAt desc. Return with keyPrefix only (never keyHash).
    - `createApiKey(name: string, createdById: string)`: Generate uuid v4 as raw key, SHA-256 hash it (crypto.createHash('sha256').update(rawKey).digest('hex')), extract last 4 chars as prefix, store LicenseApiKey record. Return { rawKey, keyPrefix, id } — rawKey shown ONCE.
    - `revokeApiKey(id: string)`: Update LicenseApiKey isActive=false, revokedAt=now.

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-04 | Tampering | LicenseController POST /activate | mitigate | JWT verified with RS256 public key — adversary cannot forge valid signature without private key. jwt.verify() checks exp and algorithm automatically. |
| T-05-05 | Spoofing | LicenseApiKeyGuard | mitigate | API key hashed with SHA-256 at rest. Key prefix (last 4 chars) only for display — full hash comparison on each request. Revocation supported via isActive flag. |
