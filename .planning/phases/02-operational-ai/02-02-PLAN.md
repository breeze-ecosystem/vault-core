<objective>
Extend incident management with evidence attachment and closure report generation. By the end of this plan, operators can attach video clips, snapshots, and access event references to incidents, view all evidence in a dedicated section, and generate/download PDF closure reports with full incident timeline, evidence, and actions taken.

Purpose: Evidence attachment completes the incident investigation workflow — operators need to collect and preserve proof of events. Closure reports provide the formal documentation needed for compliance, reporting, and post-incident review. Together they deliver INC-03 and INC-06.

Output: Evidence attachment API + dashboard UI and PDF closure report generation with template.
</objective>

    ```typescript
    export async function downloadIncidentReport(incidentId: string): Promise<void> {
      const res = await fetchWithAuth(`${API_URL}/api/incidents/${incidentId}/report`);
      if (!res.ok) throw new Error('Erreur lors du téléchargement du rapport');
      const blob = await res.blob();
      const url = window.URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url; a.download = `incident-${incidentId.substring(0,8)}.pdf`;
      a.click(); window.URL.revokeObjectURL(url);
    }
    ```
